PC 가이드
섹션	                높이(px)	    내부 여백	            구성 요소 위치 예시
Header	            80px	    좌우 40px	            로고 / 메뉴 / 로그인 버튼
Hero	            800~1000px	가운데 정렬	            제목 + 보조텍스트 + CTA
About + Services	800px	    2단 구성 (720px/720px)	왼쪽 텍스트, 오른쪽 카드 4개 (2x2)
Portfolio Slider	600px	    좌우 60px	            가로 슬라이드 카드 (썸네일+텍스트)
Contact Form	    500px	    가운데 정렬 (500px width)	제목 input + textarea + 버튼
Notice	            400px	    1열 또는 2열	            리스트 3~4개 + 전체보기 버튼
Footer	            200px	    3열 구성	                회사정보 / 링크 / 소셜

모바일 스타일 가이드
섹션              | 높이(px)     | 구성 방식
Header           | 64px         | 햄버거 메뉴 + 로고
Hero             | 600~700px    | 가운데 정렬, 텍스트 줄수 제한
About/Services   | 800px        | 카드 1열, 회사소개 텍스트 위
Portfolio        | 400~600px    | 1~2개씩 보이는 슬라이더
Contact / Notice | 500~600px    | 세로형 정렬, 버튼 w-full
Footer           | 150px        | 세로 정렬, 중앙 정렬

기본 스타일 가이드
요소      | 값
폰트      | Noto Sans KR / Inter / Pretendard
H1 제목   | 48px / Bold / Line-height: 1.2
H2 제목   | 32px / Semibold
본문      | 18px / Regular / 회색톤: #C0C0C0
포인트컬러 | #00C6FF (버튼, 강조링크 등)
버튼 높이  | 48px / radius: 8px
카드 배경  | #1E1E1E / rounded + 그림자
배경 전체  | #0F172A 또는 #111111
